Detailed Description

The max_weighted_independent_set() finds the maximum weighted set based on provided nice tree decomposition and stores it in provided property map colors. The total weight of the found set is the return value of the function.

[in]gAn undirected graph. The graph type must be a model of VertexListGraph.
[in]dAn undirected graph containing the nice tree decomposition of graph g. The graph type must be a model of VertexListGraph.
[in]bagsThe bags property map. The type must be a model of a mutable Readable Property Map. The key type of the map must be a vertex descriptor of d. The value type of the map must be a model of Mutable_Container. It also must implement function insert([position/hint iterator], [element]). Container value type must be vertex descriptor of g.
[in]rootVertex descriptor of the root of d.
[in]weightsProperty map representing weights of vertices of graph g. The type must be a model of a mutable Readable Property Map. The key type of the map must be a vertex descriptor of g. The value type of the map must be an unsigned integer.
[out]colorsProperty map used by the algorithm to represent the maximum weighted set. At the end of algorithm, the white vertices define the maximum weighted set. The type must be a model of a mutable ReadWrite Property Map. The key type of the map must be a vertex descriptor of g. The value type of the map must be a model of ColorValue.
