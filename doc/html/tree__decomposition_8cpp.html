Function for constructing tree decomposition of graph.

Based on given graph g it constructs tree decomposition, that will be stored in graph d. Bags for each node will be stored in property map bags. Decomposition itself will be of width at most 3k+4 - if such decomposition is not possible to construct, function will return false

Template Parameters
Graphtype of graph for which the tree decomposition will be constructed
Decompositiontype of graph of tree decomposition
Bagsboost PropertyMap in which will be stored bags for each node of decomposition d
Parameters
[in]ggraph whose tree decomposition will be constructed
[out]dgraph in which will be the decomposition stored
[out]bagsPropertyMap containing bags for each node of d
[in]kparameter, which will limit width of decomposition's bags
Returns
true if tree decomposition of graph g exists (with width at most 3k+4), false if not
